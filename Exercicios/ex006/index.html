Principais formatações

Negrito/Destaque

Para digitar uma palavra em negrito se deve usar a tag <strong>, desse modo você falará para o programa que aquela palavra é uma palavra de destaque no texto, e não só muda a forma da palavra, ex: Negrito

Itálico/Enfase

Para digitar uma palavra em itálico se deve usar a tag <em>, dessa forma também mostrará para o programa que aquela palavra tem um significado importante para o texto

Vale ressaltar que tem outras formas de colocar as palavras em negrito e italico, porém são formas que alterão só a forma da palavra(O visual) e não o sentido ou significado daquela palavra

Texto marcado

Para marcamos um texto usamos a tag <mark>, para muda a cor dessa marcação usamos uma tag do CSS, a tag <style> mark{} </style> dentro de Head

ou usamos dentro da tag Mark, ex: Yonnara é linda

Texto pequeno

Para colocar um texto pequeno usamos a tag <small>, para utilizar texto grande usamos big, porem a tag tá obsoleta. Ex: Essa letra é pequena

Texto deletado

Para inserirmos um texto deletado usamos a tag <del>, ex: Texto deletado.

Texto inserido

Para digitarmos um texto inserido usamos a tag <ins>, ex: Texto inserido

Texto Sobrescrito

Para inserirmos um texto sobreescrito usamos a tag <sup>, ex: x^2+3 vira x 2+3

Texto subscrito

Para colocarmos um texto subscrito usamos a tag <sub>, ex: H20 em H 2O
